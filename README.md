# alt:V Module SDK

[Documentation](https://docs.altv.mp/sdk/index.html)

## Naming style

### Enums

`PascalCase` for name and `UPPER_CASE` for enumerators, for example:
```cpp
enum class AmmoSpecialType : uint32_t
{
  NONE,
  ARMOR_PIERCING,
  EXPLOSIVE,
  FULL_METAL_JACKET,
  HOLLOW_POINT,
  INCENDIARY,
  TRACER
};
```
